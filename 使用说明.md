# 文件整理工具使用说明

> 新版默认语言为英语，并支持在图形界面切换 English、简体中文、Français、Deutsch、日本語；命令行可使用 `--lang` 指定语言。

## 功能简介
这个工具可以帮你：
- ✅ 扫描文件夹内所有文件（包括子文件夹）
- ✅ 按文件格式自动分类（图片、视频、文档等）
- ✅ 智能处理重名文件（自动添加数字后缀如 `filename_1.jpg`）
- ✅ 提供**复制模式**（保留原文件，更安全）
- ✅ 提供**移动模式**（整理后删除原文件，节省空间）
- ✅ 生成详细整理报告

## 使用方法

### 方法1：图形界面版（推荐）
1. 双击 `图形界面整理.bat` 文件
2. 点击"添加文件夹"按钮，选择要整理的文件夹（可添加多个）
3. 点击"选择文件夹"按钮，选择目标文件夹
4. 选择操作模式：复制模式（安全）或移动模式（整理后清空源文件夹）
5. 点击"开始整理"按钮
6. 等待整理完成，查看日志和报告

### 方法2：命令行版（适合高级用户）
1. 双击 `一键整理.bat` 文件
2. 按提示输入文件夹路径
3. 选择操作模式
4. 确认后开始整理

### 方法3：命令行运行
```bash
python file_organizer.py      # 命令行版本
python file_organizer_gui.py  # 图形界面版本
```

## 操作流程

```
1. 输入文件夹路径 → 2. 选择模式 → 3. 确认 → 4. 自动整理 → 5. 查看报告
```

### 步骤说明

**步骤1：输入文件夹路径**
- 输入要整理的文件夹路径（例如：`C:\Users\Downloads`）
- 直接回车使用当前文件夹

**步骤2：选择操作模式**
- 输入 `1`：复制模式（推荐，保留原文件）
- 输入 `2`：移动模式（整理后删除原文件，节省空间）

**步骤3：确认**
- 输入 `y` 确认开始整理
- 输入 `n` 取消操作

## 文件分类规则

整理后的文件会按以下结构存放：

```
organized/
├── images/          # 图片文件
│   ├── jpg/
│   ├── png/
│   └── ...
├── videos/          # 视频文件
│   ├── mp4/
│   ├── avi/
│   └── ...
├── documents/       # 文档文件
│   ├── pdf/
│   ├── docx/
│   └── ...
├── audios/          # 音频文件
├── archives/        # 压缩包
├── executables/     # 可执行文件
├── code/            # 代码文件
└── others/          # 其他文件
```

## 重名文件处理

如果遇到重名文件，会自动重命名：
```
photo.jpg      → photo.jpg
photo.jpg      → photo_1.jpg
photo.jpg      → photo_2.jpg
```

## 整理报告

整理完成后会在 `organized` 文件夹中生成 `organizer_report.txt`，包含：
- 整理时间
- 文件统计信息
- 文件类型分布
- 每个文件的详细处理记录

## 注意事项

⚠️ **重要提醒：**
- 首次使用建议使用**复制模式**（输入 `1`），保留原文件
- 确认整理结果无误后，再手动删除原文件
- 整理过程中请勿关闭程序
- 大文件夹可能需要较长时间，请耐心等待

## 常见问题

**Q: 整理后可以恢复吗？**
A: 使用复制模式可以，原文件保留；使用移动模式则无法自动恢复。

**Q: 支持哪些文件格式？**
A: 支持常见格式（图片、视频、音频、文档等），未知格式会放入 `others` 文件夹。

**Q: 会处理子文件夹吗？**
A: 会，会扫描所有子文件夹中的文件。

**Q: 会删除我的文件吗？**
A: 不会，工具只移动或复制文件，不会删除。但移动模式会从原位置移动文件。

## 技术支持

如有问题，请查看 `organizer_report.txt` 报告文件，或重新运行程序。
